# 🎣 FishPulse

**Открытая система рыболовного интеллекта. Бесплатно для каждого рыбака.**

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Go Version](https://img.shields.io/badge/Go-1.25+-00ADD8?logo=go)](https://golang.org)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](https://github.com/nurtidev/fishpulse/issues)

> [English](README.md) · **Русский** · [Қазақша](README_KZ.md)

FishPulse рассчитывает **Индекс Клёва (0–100)** для любой точки на карте в реальном времени, объединяя лунно-солнечную теорию с живыми метеорологическими данными. Без подписок, без платного оборудования.

> Вдохновлено прогнозом клёва Garmin — создано, чтобы быть бесплатным для всех.

**[→ Открыть приложение](https://fishpulse-production.up.railway.app)**

## Как работает Индекс Клёва

```
Индекс = (Солунар × 0.25) + (Давление × 0.30) + (Температура × 0.20)
        + (ВремяСуток × 0.15) + (Ветер × 0.10)
        × СезонныйМножитель
```

| Фактор | Вес | Источник |
|--------|-----|---------|
| Солунар (положение луны/солнца) | 25% | Астрономический расчёт |
| Тренд атмосферного давления | 30% | Open-Meteo API (бесплатно) |
| Температура воды | 20% | Производная от температуры воздуха |
| Время суток | 15% | Расчёт рассвета/заката |
| Скорость ветра | 10% | Open-Meteo API (бесплатно) |

**Ключевой инсайт:** Рыба чувствует изменения давления через плавательный пузырь. Падающий фронт давления = агрессивное кормление перед бурей. Именно этот сигнал пропускают большинство приложений.

Итоговая оценка: **Плохо** (0–39) · **Нормально** (40–59) · **Хорошо** (60–79) · **Отлично** (80–100)

## Возможности

- **Карта с кликом** — нажми на любую реку, озеро или море, чтобы получить Индекс Клёва
- **48-часовой график** — почасовой прогноз клёва
- **Солунарные окна** — главные и второстепенные периоды с точным временем начала и конца
- **Профили рыб** — Щука, Окунь, Карп, Лещ, Сом, Судак
- **Предупреждение об ареале** — если вид редко встречается в выбранном регионе
- **ИИ-совет рыбаку** — практичные рекомендации от Claude на русском, казахском или английском
- **Мультиязычный** — Русский 🇷🇺 · Казахский 🇰🇿 · Английский 🇬🇧
- **Mobile-first** — работает на любом телефоне без установки приложения

## Быстрый старт

### Требования
- Go 1.25+
- Node.js 20+

### Запуск локально

```bash
git clone https://github.com/nurtidev/fishpulse.git
cd fishpulse

# 1. Запуск бэкенда (порт 8080)
go run ./cmd/server/

# 2. В другом терминале — запуск фронтенда (порт 3000)
cd web
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000).

### Запуск через Docker

```bash
# Бэкенд
docker build -t fishpulse-api .
docker run -p 8080:8080 \
  -e ALLOWED_ORIGIN=http://localhost:3000 \
  fishpulse-api

# Фронтенд
cd web
docker build -t fishpulse-web .
docker run -p 3000:3000 \
  -e NEXT_PUBLIC_API_URL=http://localhost:8080 \
  fishpulse-web
```

curl "http://localhost:8080/api/v1/bite?lat=51.18&lon=71.45&species=pike&lang=ru"

## Архитектура

```
fishpulse/
├── cmd/server/        # Точка входа
├── core/              # Движок расчёта на Go
│   ├── index.go       # Формула Индекса Клёва
│   ├── solunar.go     # Позиция луны/солнца и фаза
│   ├── solunar_periods.go  # Солунарные окна
│   ├── pressure.go    # Скоринг давления
│   ├── temperature.go # Расчёт температуры воды
│   ├── weather.go     # Open-Meteo клиент (с кэшем)
│   ├── advice.go      # Интеграция с Claude AI
│   └── loader.go      # Загрузчик конфигов видов рыб
├── api/               # REST API сервер
│   ├── server.go      # HTTP сервер
│   ├── handlers.go    # Эндпоинты
│   └── middleware.go  # CORS, rate limiting, логирование
├── web/               # Next.js 16 фронтенд
│   ├── app/           # App Router страницы
│   └── components/    # React компоненты
└── algorithms/
    ├── species/        # Конфиги видов рыб (JSON)
    └── regions/        # Региональная калибровка (JSON)
```

## Источники данных

Все источники **бесплатны и открыты**:

- **[Open-Meteo](https://open-meteo.com/)** — прогнозы погоды и давления (API-ключ не нужен)
- **[Nominatim / OpenStreetMap](https://nominatim.org/)** — геокодирование для поиска мест
- **[Anthropic Claude](https://anthropic.com/)** — ИИ-советы (опционально, нужен API-ключ)

## Участие в разработке

FishPulse развивается сообществом. Вот где нужна помощь:

**Добавить вид рыбы** — создай `algorithms/species/yourfish.json` с сезонными и температурными коэффициентами.

**Добавить регион** — создай `algorithms/regions/yourregion.json` с данными калибровки для рек и озёр твоего региона.

**Улучшить алгоритм** — формула клёва в [core/index.go](core/index.go). Открой PR с данными и обоснованием.

**Исправить баг или добавить фичу** — смотри [Issues](https://github.com/nurtidev/fishpulse/issues).
