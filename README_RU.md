FishPulse рассчитывает **Индекс Клёва (0–100)** для любой точки на карте в реальном времени, объединяя лунно-солнечную теорию с живыми метеорологическими данными. Без подписок, без платного оборудования.

> Вдохновлено прогнозом клёва Garmin — создано, чтобы быть бесплатным для всех.

## Как работает Индекс Клёва

Индекс рассчитывается из 5 взвешенных факторов:

```
Индекс = (Солунар × 0.25) + (Давление × 0.30) + (Температура × 0.20)
        + (ВремяСуток × 0.15) + (Ветер × 0.10)
```

Затем применяется сезонный множитель в зависимости от вида рыбы.

| Фактор | Вес | Источник |
|--------|-----|---------|
| Солунар (положение луны/солнца) | 25% | Астрономический расчёт |
| Тренд атмосферного давления | 30% | Open-Meteo API (бесплатно) |
| Температура воды | 20% | Производная от температуры воздуха |
| Время суток | 15% | Расчёт рассвета/заката |
| Скорость ветра | 10% | Open-Meteo API (бесплатно) |

**Ключевой инсайт:** Рыба чувствует изменения давления через плавательный пузырь. Падающий фронт давления = агрессивное клёвание перед бурей. Это сигнал, который большинство приложений упускают.

## Возможности

- **Интерактивная карта** — кликни на любую реку, озеро или море, чтобы получить Индекс Клёва
- **48-часовой график** — почасовой прогноз клёва
- **Календарь на 7 дней** — лучшие дни недели с первого взгляда
- **Профили рыб** — Щука, Окунь, Карп, Форель (больше — от сообщества)
- **Региональная калибровка** — локальные коэффициенты для разных водоёмов
- **Открытый API** — интеграция с Garmin, Apple Watch, WearOS и любым приложением

## Быстрый старт

### Требования
- Go 1.22+
- Node.js 20+

### Запуск API локально

```bash
git clone git@github.com:nurtidev/fishpulse.git
cd fishpulse

# Запуск бэкенда
go run ./cmd/

# Запрос к API
curl "http://localhost:8080/api/bite?lat=51.1801&lon=71.4460&species=pike"
```

### Пример ответа

```json
{
  "location": { "lat": 51.1801, "lon": 71.4460 },
  "species": "pike",
  "current": {
    "index": 78,
    "label": "Good",
    "reason": "Давление падает перед дождём — щука кормится агрессивно"
  },
  "best_window": {
    "start": "2026-03-02T06:00:00Z",
    "index": 91
  }
}
```

## Источники данных

Все источники данных **бесплатны и открыты**:

- **[Open-Meteo](https://open-meteo.com/)** — прогнозы погоды и давления (API-ключ не нужен)
- **[Astronomy Engine](https://github.com/cosinekitty/astronomy)** — расчёты луны/солнца
- **[OpenStreetMap](https://www.openstreetmap.org/)** / **Leaflet** — карты

## Участие в разработке

FishPulse развивается сообществом. Вот где нужна ваша помощь:

### Добавить вид рыбы
Создайте `algorithms/species/yourfish.json` с сезонными и температурными коэффициентами.

### Добавить регион
Создайте `algorithms/regions/yourregion.json` с данными калибровки для рек и озёр вашего региона.

### Улучшить алгоритм
Формула клёва находится в [core/index.go](core/index.go). Если у вас есть данные, подтверждающие лучшие веса — откройте PR с обоснованием.
