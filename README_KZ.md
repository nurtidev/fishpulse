FishPulse кез келген нүкте үшін **Шабу Индексін (0–100)** нақты уақытта есептейді — ай-күн теориясын тірі метеорологиялық деректермен біріктіреді. Жазылымсыз, ақылы жабдықсыз.

> Garmin балық аулау болжамынан шабыт алынды — барлығына тегін болу үшін жасалды.

## Шабу Индексі қалай жұмыс істейді

Индекс 5 өлшенген факторdan есептеледі:

```
Индекс = (Солунар × 0.25) + (Қысым × 0.30) + (Температура × 0.20)
        + (ТәуліктіңУақыты × 0.15) + (Жел × 0.10)
```

Содан кейін балық түріне байланысты маусымдық коэффициент қолданылады.

| Фактор | Салмақ | Дерек көзі |
|--------|--------|-----------|
| Солунар (ай/күн орны) | 25% | Астрономиялық есеп |
| Атмосфералық қысым үрдісі | 30% | Open-Meteo API (тегін) |
| Су температурасы | 20% | Ауа температурасынан туынды |
| Тәуліктің уақыты | 15% | Таң/кеш есебі |
| Жел жылдамдығы | 10% | Open-Meteo API (тегін) |

**Негізгі түсінік:** Балық қысым өзгерістерін жүзу қабы арқылы сезеді. Қысымның төмендеуі = дауылға дейін агрессивті тамақтану. Бұл сигналды көптеген қосымшалар жіберіп алады.

## Мүмкіндіктер

- **Интерактивті карта** — кез келген өзен, көл немесе теңізді басып, Шабу Индексін алыңыз
- **48 сағаттық кесте** — сағаттық шабу болжамы
- **7 күндік күнтізбе** — аптаның ең жақсы күндері бір қарауда
- **Балық профильдері** — Шортан, Алабұға, Табан, Форель (қосымша — қауымдастықтан)
- **Аймақтық калибрлеу** — әр су қоймасына арналған жергілікті коэффициенттер
- **Ашық API** — Garmin, Apple Watch, WearOS және кез келген қосымшамен интеграция

## Жылдам бастау

### Талаптар
- Go 1.22+
- Node.js 20+

### API-ді жергілікті іске қосу

```bash
git clone git@github.com:nurtidev/fishpulse.git
cd fishpulse

# Бэкендті іске қосу
go run ./cmd/

# API сұрауы (Астана, Есіл өзені)
curl "http://localhost:8080/api/bite?lat=51.1801&lon=71.4460&species=pike"
```
